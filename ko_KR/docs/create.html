Hooks 생성하기

GORM은 BeforeSave, BeforeCreate, AfterSave, AfterCreate Hooks를 허용합니다. 해당 메서드는 레코드를 만들 때 호출됩니다. 자세한 내용은 Hooks를 참조하십시오.

일괄 삽입

Create 메서드에 슬라이스 데이터를 전달하면 GORM은 모든 데이터를 삽입하고 기본 키를 채우기위한 단일 SQL 문을 생성하며, Hooks 메서드또한 호출됩니다.

일괄 삽입은 Upsert 및 Create With Associations에도 지원됩니다.

SQL Expr/Context Valuer로 생성

GORM을 사용하면 SQL 표현식을 사용하여 데이터를 삽입 할 수 있습니다.이 목표를 달성하는 데는 두 가지 방법이 있습니다. map [string] interface {} 또는 사용자 정의 데이터 유형에서 생성합니다.

Upsert / On Conflict

GORM은 서로 다른 데이터베이스에 대해 호환 가능한 Upsert 지원을 제공합니다.

또한 고급 쿼리 문서에서 FirstOrInit, FirstOrCreate에 대하여 확인하십시오.

자세한 내용은 Raw SQL 및 SQL Builder를 확인하십시오.
